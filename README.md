💰 Cofrinho em Java

Este projeto foi desenvolvido como parte dos meus estudos em Java, com foco em Programação Orientada a Objetos (POO).

🚀 Funcionalidades
Adicionar moedas (Real, Dólar e Euro)
Remover moedas
Listar moedas armazenadas
Calcular o total convertido em reais

🧠 Conceitos aplicados
Herança
Polimorfismo
Classes abstratas
Encapsulamento
Estrutura de dados (ArrayList)

💻 Tecnologias
Java
Eclipse

🎯 Objetivo
Praticar conceitos fundamentais de POO e desenvolver lógica de programação através de um sistema simples e funcional.

Este projeto faz parte da minha transição para a área de tecnologia e representa um passo importante na minha evolução como desenvolvedora.
